
# Pong Clone

Este jogo é um clone do PONG de Atari, foi desenvolvido com Unity3D para ser jogado em um PC.

## Instalação

Clone o projeto

```bash
  git clone git@github.com:LeoFeitosa/PongClone.git
  abra o projeto na engine Unity3D
```
